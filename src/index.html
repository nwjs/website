NW.js (previously known as node-webkit) lets you call all Node.js modules directly from DOM and enables a new way of writing applications with all Web technologies.

  • New way of writing native applications web technologies: HTML5, CSS3, and WebGL

  • Full support for the features in browser

  • Complete support for Node.js APIs and all third party modules

  • Call Node.js modules directly from the DOM

  • JavaScript Source Protection

  • Available on Linux, Mac OS X and Windows
